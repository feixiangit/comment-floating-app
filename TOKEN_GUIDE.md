# GitHub上传 - 使用Personal Access Token

## 步骤1：生成Token

1. 访问：https://github.com/settings/tokens/new
2. 输入Token名称：`comment-floating-app-upload`
3. 勾选权限：**repo** (完整控制私有仓库)
4. 点击 **Generate token**
5. **复制Token**（只显示一次！）

## 步骤2：创建仓库

1. 访问：https://github.com/new
2. 仓库名：`comment-floating-app`
3. 描述：`Comment Assistant - Floating Window App for HarmonyOS/Android`
4. 选择 **Public**
5. 点击 **Create repository**

## 步骤3：告诉我Token

把生成的Token发给我，格式：
```
ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

我会用Token自动上传代码。

---

## 为什么需要Token？

- 无需输入密码
- 更安全（可随时撤销）
- 自动化脚本可用
